# Challenge App
##### Simple app to write 10m record into clickhouse database

####requirement

+ Node.js `>=8`

+ Clickhouse
#### installation

```
git clone git@github.com:saeedjalalisj/challenge.git
npm install
```
#### usage
```$xslt
npm start
```

#### Analyse
How long all data was written to db?

